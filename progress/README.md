# Progresso dos agentes

Esta pasta guarda registros curtos de continuidade.

Cada agente deve criar ou atualizar um arquivo com o padrão:

```text
progress/YYYY-MM-DD_AGENT_PLAN_XX.md
```

Exemplo:

```text
progress/2026-06-12_DEEPSEEK_PLAN_01.md
```

Conteúdo recomendado:

- Plano atual.
- Resumo curto.
- Tarefas concluídas.
- Arquivos alterados.
- Testes executados.
- Bloqueios ou pendências.
- Próximo passo.

Mantenha o registro objetivo. Ele deve ajudar outro agente a retomar o trabalho sem ler histórico de conversa.
